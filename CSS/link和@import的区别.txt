CSS的引入方式
1、内联样式
<div style="color: #eee"></div>
2、嵌入样式
<style>
    .box {
        height: 100px;
    }
</style>
3、外部引入
<link rel="stylesheet" href="./css.css">
4、导入样式
<style>
    @import url("./css.css");
</style>

区别
1、link是一个html标签，作用不止局限于引入CSS，而@import只能用来加载CSS
2、link引入CSS时，在页面加载的同时加载，而@import在网页全部加载完成之后再加载
3、link不存在兼容性问题，而@import对于低版本浏览器不支持
